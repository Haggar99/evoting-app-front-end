# evoting-app-front-end
1. 
Install [NodeJs](https://nodejs.org/en/download/) version > 10.

2. `npm`
```bash
npm install -g @ionic/cli
```

3.
```bash
git clone https://github.com/Haggar99/evoting-app-front-end.git
```
4.
```bash
npm install
```
5.
```bash
cd evoting-app-front-end
```
6. `ionic`
```bash
ionic serve
```
